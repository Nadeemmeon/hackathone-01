# Physical AI & Humanoid Robotics Book

Welcome to the Physical AI & Humanoid Robotics educational book. This comprehensive guide covers the fundamentals of connecting artificial intelligence with physical robotics systems, focusing on humanoid robots and their control systems.

## Table of Contents

- Module 1: The Robotic Nervous System (ROS 2)
- Module 2: AI Control Systems
- Module 3: Hardware Integration
- Module 4: Advanced Topics

## Getting Started

To run this documentation locally:

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to see the documentation.
